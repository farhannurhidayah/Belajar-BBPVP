Pembuatan Formulir Pendaftaran, Tabel Daftar Mahasiswa, dan Halaman 
Biodata Menggunakan HTML (Latihan 3)

Melatih peserta dalam membuat halaman web sederhana menggunakan HTML.
Melalui tugas ini, peserta belajar membuat formulir pendaftaran, menampilkan
data dalam bentuk tabel, serta menyajikan biodata mahasiswa berdasarkan
input yang diberikan. Dengan demikian, mahasiswa dapat memahami
